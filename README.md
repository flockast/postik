# Postik

Basic API for a blog, based on a clean architecture

Built with Fastify, Postgress, Kysely, Typebox, Swagger

## Installation
```sh
npm install
```

## Development
```sh
npm run dev
```

## Start database
```sh
npm run db:up
```

## Shutdown database
```sh
npm run db:down
```

## Migration
```sh
npm run db:migrate
```

## Generate database types
```sh
npm run generate:db:types
```

## Build
```sh
npm run build
```

## Preview
```sh
npm run start
```
